Welcome to EurekaimerOS! This repository contains my personal declarative system configuration for NixOS. 

### 📝 The Story So Far
This repository is a complete, from-scratch rewrite of my previous NixOS setup. I finally decided to step up and take back control of my system. In short: **I successfully migrated this project from "spaghetti garbage generated by AI vibe-coding" to "artisanal, hand-crafted garbage written by myself."** 🎉

### ✨ Key Features

* **Dual Desktop Environments:**
    * **KDE Plasma:** Kept as a stable, full-featured fallback environment.
    * **Niri:** A scrollable-tiling Wayland compositor. This is my daily driver now, configured with buttery-smooth animations and custom keybinds.
* **True Modular Configuration:** Nix logic and native configurations are now strictly separated. Native config files (like `.kdl`, `.jsonc`, `.css`) are neatly organized in the `config/` directory and linked via `xdg.configFile`, keeping the `.nix` files clean and purely declarative.
* **Custom UI & Tooling:**
    * **Bar:** Waybar (with custom sliding volume controls & workspaces).
    * **Launcher & Menus:** Rofi-wayland.
    * **Wallpaper Engine:** `swww` with custom Bash scripts for random-transition wallpaper switching.
    * **Workflow:** `kitty` terminal, `grim` + `slurp` + `wl-clipboard` for seamless screenshots.
